# Brandopoly Blitz

**Brandopoly Blitz** — это динамичная многопользовательская экономическая настольная игра, вдохновленная классической «Монополией», но выведенная на новый уровень.

- **Прямоугольное поле 14×8** с уникальными углами: «Стартап‑хаб», «Фондовый рынок», «Регулятор», «Глобальная сцена».
- **Топ‑бренды мира** в виде текстовых клеток с безопасными плейсхолдерами.
- **Синергии отраслей** и динамическая экономика брендов.
- **Аукционы с неполной информацией**, контракты, альянсы и карточки «Инновации и Регуляции».
- **Быстрый режим 45–60 минут** и мета‑прогресс сезонов.

Игра подойдет:
- новичкам благодаря интерактивному обучению;
- соревновательным игрокам из‑за рейтинговой системы и глубокой экономики;
- компаниям друзей за счет кооперативных альянсов и быстрых сессий.

> **По умолчанию активирован "Generic mode" — все бренды отображаются нейтрально. Для использования реальных названий необходимы лицензии.**
