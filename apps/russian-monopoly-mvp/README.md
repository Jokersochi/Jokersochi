# Русская Монополия (офлайн прототип)

## Запуск
Откройте файл `index.html` в браузере (Chrome, Firefox, Edge) в режиме file://. Дополнительных зависимостей или серверов не требуется.

## Горячие клавиши
- **R** — бросок кубиков текущим игроком.
- **B** — покупка доступного актива.
- **E** — завершение хода.
- **L** — смена языка интерфейса по циклу.

## Пресеты
В комплекте два пресета:
- `rus` — классический баланс с акцентом на российские города и карты с культурными референсами.
- `intl-friendly` — более мягкие налоги и упрощённые штрафы, подходит для коротких партий.

Доступно 2–6 игроков, управление полностью с одного устройства.
