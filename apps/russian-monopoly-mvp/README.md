# Русская Монополия — React + Vite прототип

## Быстрый старт
1. Установите зависимости: `npm install`
2. Запустите локальный сервер разработки: `npm run dev`
3. Сборка production-версии: `npm run build`
4. Просмотр собранной версии: `npm run preview`
5. Запуск unit-тестов: `npm run test`

> Примечание: проект использует только системные шрифты и обобщённые приёмы верстки, без брендинга и защищённых материалов ru.monopoly-club.com. Внешние подключения не требуются.

## Tailwind CSS
Пользовательские переменные определены в `src/index.css` и доступны в Tailwind через алиасы:

```css
:root {
  --color-bg: #f4f6fb;
  --color-surface: #ffffff;
  --color-accent: #3b82f6;
  --color-muted: #e2e8f0;
  --color-success: #22c55e;
  --color-danger: #ef4444;
  --color-text: #0f172a;
  --color-text-muted: #475569;
}
```

Конфигурация Tailwind хранится в `tailwind.config.cjs`, а PostCSS — в `postcss.config.cjs`. Стандартные плагины: `tailwindcss` и `autoprefixer`.

## Системные шрифты
Во всём интерфейсе применяются системные гарнитуры: `'Segoe UI', 'Roboto', 'Helvetica Neue', 'Arial', 'Noto Sans', sans-serif`.

## Архитектура
- **Стек**: React 18 + TypeScript + Vite, Tailwind CSS, zustand с persist/immer для бизнес-логики.
- **Структура**: исходники в `src/`, компоненты в `src/components/`, состояние в `src/store/`, конфигурация/типы в `src/config/`.
- **Механики**: поддержка 2–6 игроков, броски кубиков, покупка/аренда собственности, налоги, карты шанс/испытание, обработка тюрьмы, микро-ивенты и контракты, журнал событий, автосохранение прогресса.
- **Данные**: JSON-файлы в `/data` (поле, колоды, локализации, пресеты, микро-ивенты, контракты). При ошибках загрузки данные подменяются на минимальные `fallbackData`.

## Доступность и управление
- Навигация с клавиатуры: таб-циклы, `focus-visible` стили, aria-атрибуты у модальных окон и списков.
- Панель управления предоставляет явные состояния и озвучиваемые подсказки.
- Модальное окно карт получает автофокус и закрывается по кнопке.

## Нереализованные функции (v2)
- Торги между игроками и аукционы (требует отдельного UI).
- Строительство домов/отелей и динамическая аренда.
- Расширенная экономика контрактов (автоматическое начисление).
- Полноценная локализация текста игры (сейчас применяются fallback-строки).

## Допущения и решения
- Стартовый капитал равен зарплате за проход старта выбранного пресета.
- Тюрьма оплачивается автоматически (штраф или карта освобождения).
- При отсутствии JSON-файлов игрок получает минимальный набор данных.
- Все токены — SVG-плейсхолдеры, соответствующие названию, без лицензируемых образов.
- Интерфейс вдохновлён публичной компоновкой ru.monopoly-club.com, но создан с нуля и без фирменных элементов.
